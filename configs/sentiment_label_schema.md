# Sentiment Label Schema

## 负面

整体表达不满、抱怨、失望、不推荐或体验差。

示例：

- 客服回复倒是快，就是问题完全没解决。
- 风景还可以，可排队体验太糟糕了。
- 味道不是不能吃，但确实不会再点了。

## 中性

整体是事实陈述、普通体验、弱情绪或态度不明显；没有明确推荐或抱怨。

示例：

- 包装基本完整，物流比预计晚了一天。
- 课程内容比较基础，适合先了解一下。
- 这家店位置挺好，菜量属于正常水平。

## 正面

整体表达满意、认可、喜欢、推荐或体验好；可以包含轻微小缺点，但最终态度为正面。

示例：

- 价格不算便宜，但体验确实值这个价。
- 人多了一点，不过景色真的很惊艳。
- 刚开始不适应，后面越学越有收获。

## Hard Set 设计原则

Hard Set 聚焦弱情绪、转折表达和边界样本，不追求简单强情绪样本。
